Support

Need help with Thread?

If something looks off, a puzzle feels broken, or your app is behaving strangely, send us a message and we’ll investigate.

Contact support

Email: support@daily-thread.co

When reporting an issue, include the device you used, whether it happened on iPhone or iPad, the app version if known, and a screenshot if you have one.

Common questions

When does the daily puzzle reset?

Thread’s daily puzzle is designed to refresh at midnight in the app’s configured daily schedule. If you leave the app open, it should refresh automatically when the new day starts.

Can I play offline?

Yes. Thread is built to be local-first. The main gameplay does not depend on a network connection.

How do reminders work?

If you turn them on, Thread can send local reminders at 9:00 AM and 9:00 PM. You can change this in the app’s Settings.

How do I reset my progress?

The app includes reset controls in Settings for local progress and tutorial flow. If something still looks wrong after a reset, email support with details.

How do I report a puzzle issue?

Email support with the puzzle number or date, what answer you expected, and why you think the clue set was unclear or incorrect.
