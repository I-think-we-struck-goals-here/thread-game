## Description

Daily Thread is a daily word puzzle about finding the one word that connects five clues.

Start with one clue and guess whenever you feel ready. Every wrong guess reveals the next clue, so fewer clues means a better score.

Features:

- One new daily puzzle
- Practice rounds to learn the format
- Streaks, history, and local stats
- Result sharing
- Clean, focused design
- Works offline
- Optional local reminders
- Private iCloud sync for your own devices
